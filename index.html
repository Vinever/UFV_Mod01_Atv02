Para finalidade de conclusão da atividade, segue os exemplos das TAGs mencionadas em aula:

  • BACKGROUND: define imagem de fundo em uma página.
  • TEXT: define a cor do texto da página web.
  • LINK: define a cor dos links da página web (padrão é azul).
  • BGCOLOR: define a cor de fundo da página (padrão é branca).

  • LEFT: alinha o texto à esquerda

  • CENTER: centraliza o texto

  • RIGHT: Alinha o texto à direita

  • Lista ordenada
    1. Para gerar listas ordenadas, usa-se a tag "ol"(de ordered list)
    2. Ele delimita do primeiro até o último item da lista ordenada.
    3. Cada item da lista ordenada deve ser precedido da tag "li"(de line item).
    4. E ela pode ser personalizada com os atributos TYPE, START ou VALUE.
